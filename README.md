# 🧠 Mental Health Chatbot (Python)

This project is a simple chatbot that interacts with users and detects emotions using keyword-based analysis.

## Features
- Emotion detection using smart keywords
- Motivation generator
- User name memory
- Mood tracking system
- Mood visualization using graphs
- Mood data saved to file

## Technologies Used
- Python
- Google Colab
- Matplotlib

## How It Works
The chatbot takes user input and matches keywords to detect emotions like stress, sadness, or happiness. Based on this, it responds with helpful suggestions and tracks mood data.

## Future Improvements
- Add machine learning for better emotion detection
- Create a web interface
- Store data in a database
